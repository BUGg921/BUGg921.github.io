# 关于我

你好，我是 **BUGg921**。

我喜欢把技术、想法和日常的小麻烦，变成真正能被使用的网页与工具。这个网站会跟着我一起成长：现在是一张个人主页，接下来会逐渐增加工具、博客和项目记录。

## 这个网站

- 使用 Vue 3 与 Vite 构建
- 源码托管在 [GitHub](https://github.com/BUGg921/BUGg921.github.io)
- 通过 GitHub Pages 自动发布

界面设计来自于 [Arvin](https://github.com/6owen/arvin)，并在原项目代码基础上进行了个人化修改。
